Result

=

defn

(λ [(Ref (Result a b)), (Ref (Result a b))] Bool) 

                    (= a b)

equality is defined as the equality of both the constructor and the contained value, i.e. (Success 1) and (Success 1) are the same, but (Success 1) and (Success 2) are not.

creates a Error.

Success

template

(λ [a] (Result a b))

creates a Success.

and-then

defn

(λ [(Result a b), (λ [a] (Result c b))] (Result c b)) 

                    (and-then a f)

takes a Result and applies a function f to it if it is a Success type. In the case that it is an Error, it is returned as is.

It is thus quite similar to map, but it will unwrap the value.

apply

defn

(λ [(Result a b), (λ [a] c), (λ [b] d)] (Result c d)) 

                    (apply a success-f error-f)

takes a Result a and applies functions to them, one in the case that it is an Error, one in the case that it is a Success.

from-error

defn

(λ [(Result a b), b] b) 

                    (from-error a dflt)

is an unwrapper that will get the value from an Error. If a is a Success, a default value will be returned.

from-success

defn

(λ [(Result a b), a] a) 

                    (from-success a dflt)

is an unwrapper that will get the value from a Success. If a is an Error, a default value will be returned.

map

defn

(λ [(Result a b), (λ [a] c)] (Result c b)) 

                    (map a f)

takes a Result and applies a function f to it if it is a Success type, and wraps it back up. In the case that it is an Error, it is returned as is.

or-else

defn

(λ [(Result a b), (λ [b] (Result a c))] (Result a c)) 

                    (or-else a f)

takes a Result and applies a function f to it if it is an Error type. In the case that it is a Success, it is returned as is.

It is the inverse of and-then.

success?

defn

(λ [(Ref (Result a b))] Bool) 

                    (success? a)

checks whether the given value a is an Error.

It is the inverse of success?.

to-maybe

defn

(λ [(Result a b)] (Maybe a)) 

                    (to-maybe a)

is a function that will convert a Result to a Maybe, where a Success becomes a Just and an Error becomes a Nothing.

The error value is thrown away.

unsafe-from-error

defn

(λ [(Result a b)] b) 

                    (unsafe-from-error a)

is an unsafe unwrapper that will get the value from an Error. If a is a Success, a runtime error will be generated.

unsafe-from-success

defn

(λ [(Result a b)] a) 

                    (unsafe-from-success a)

is an unsafe unwrapper that will get the value from a Success. If a is an Error, a runtime error will be generated.

unwrap-or-else

defn

(λ [(Result a b), (λ [b] a)] a) 

                    (unwrap-or-else a f)

takes a Result and either unwraps it if it is a Success, or calls a function f on the value contained in the Error.

unwrap-or-zero

defn

(λ [(Result a b)] a) 

                    (unwrap-or-zero a)

takes a Result and either unwraps it if it is a Success, or calls zero. zero must be defined on the Success type.
